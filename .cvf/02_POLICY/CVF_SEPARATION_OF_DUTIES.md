# CVF SEPARATION OF DUTIES

## 1. ROLE CONFLICT MATRIX

| Role | Cannot |
|------|--------|
| CVF Architect | Certify agent |
| Agent Owner | Self-certify |
| Auditor | Modify agent config |
| Operator | Modify CVF core |

---

## 2. PURPOSE

Prevents governance bypass and authority concentration.

---

Violation invalidates certification.
